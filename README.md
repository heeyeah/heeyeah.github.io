## Blog
### local
`npm start` 또는 `gatsby develop`
### deploy
`npm run deploy`
`git commit & push`
> npm run deploy는 로컬에 가진 소스를 github.io 블로그로 배포할 뿐이지 github에 소스까지 push해서 저장하는 역할은 아니다. deploy를 했으면 꼭, github에 배포도 해놓을 것!
